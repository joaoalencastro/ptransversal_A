Sobre

Aplicação web destinada a alunos, monitores, professores e funcionários do departamento de Engenharia Elétrica (ENE) e, possivelmente, de outros departamentos. A aplicação permite ao usuário cadastrado fazer solicitações de reserva de salas,

gerenciar o uso das mesmas e, a todos os usuários, o sistema permite que sejam visualizados os mapas de sala. Para informações sobre o uso do sistema. Para informações de como utilizar à aplicação acesse os termos de uso. Ou se deseja adquirir o sistema, o projeto está disponibilizado neste link.
